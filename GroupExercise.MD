## Instructions for group exercise

1. Create folder with your first name in your /media/volume/MyData folder
2. Load a mouse model from in /media/share/MorphoCloudCephShare/SlicerMorph102/small_data/ and the Murat_template.mrk.json file into Slicer
3. Rename the `Murat_template` object to the match the ID of the mouse model you are landmarking
4. Collect the 12 LMs indicated (6 on both views)
5. Use the ExportAs functionality in Data module to save the pointList object to the folder you created in Step 1 (e.g., /media/volume/MyData/Murat). Create subfolder for each day (Day1, Day2, Day3).
